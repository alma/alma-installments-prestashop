Exclude Categories documentation
===================================

## Context
The function excluded categories allows you to exclude some categories to pay with Alma and display the widget.
In the Settings configuration page you can enable or disabled the message to display instead of the widget when the category is excluded,
and you can also customize the message to display instead of the widget when the category is excluded.

You have also the page excluded categories to display a grid with the list of categories and you can select the categories to exclude from this page.
You need to select the categories to exclude and click on the "Enable exclude" or "Disable exclude" to add or remove the id of categories in the array saved in the key `ALMA_EXCLUDE_CATEGORIES`.
