I18N documentation
===================================

## Context
We use the new system of translation in PrestaShop 1.7, which is based on Symfony Translation component.
This system allows us to manage translations in a more efficient way and to use the same translation files for both the front office and the back office.

### Link to the documentation
[Symfony Translation component documentation](https://devdocs.prestashop-project.org/1.7/modules/creation/module-translation/new-system/)

## Example code
### For legacy controllers
```php
$this->module->trans('My module title', [], 'Modules.Mymodule.Something');
```

### For Symfony controllers
```php
$this->trans('Some text being translated', 'Modules.Mymodule.Admin', []);
```

### For Twig templates
```twig
{{ 'Some text being translated'|trans({}, 'Modules.Mymodule.Admin') }}
{{ 'Hello %username%!'|trans({'%username%': 'John'}, 'Modules.Mymodule.Admin') }}
```

### For Smarty templates
```smarty
{l s='Welcome to this page!' d='Modules.Mymodule.Somefile'}
{l s='Hello %username%!' sprintf=['%username%' => 'John'] d='Modules.Mymodule.Somefile'}
```
