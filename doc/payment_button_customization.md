Payment button customization documentation
===================================

## Context
The payment button customization is used to customize the payment button text in the checkout page of the front office.
You can customize the title and description of each kind of payment method (Pay now, Pay later, PnX and Credit).
The payment button customization get the wording of each language handle from our module automatically if the language is enabled in the CMS and you can customize it for each language in the back office of PrestaShop.
