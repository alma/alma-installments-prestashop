Ps Accounts documentation
===================================

## Context
To get the CDN url and Presenter we check if the service is available, if not, we force the installation of PS Accounts Module. It’s the Prestashop Process for PS Accounts.

Then we display the PS Account block and if PS Account is not verified we don’t display the configuration form

If the module is already configured we don’t verify the PS Account to avoid block old merchant to update their configuration
