Refund documentation
===================================

## Context
The refund allows you to make refund of your Alma payment from the CMS PrestaShop back office.
We have two refunds process:
- The first one, allows you to make a full refund of the payment from the changement state of the order.
    You need to enable this option in the configuration page and select the state of the order that will trigger the refund process.
    The list of state order is get from the state order of the PrestaShop, so you can create a new state order if you want to have a specific state for the refund process.
    By default, this options is disabled.
- The second one allows to make a total or partial refund from the order details page,
    you can select the products and the quantity that you want to refund, and the refund will be made for the selected products.
